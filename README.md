# Amazon Sales Dashboard using Looker Studio

## Overview
This interactive dashboard was built using Looker Studio to analyze Amazon sales data. It includes key visualizations that help track performance metrics such as total sales, profit, product categories, and more.

## Key Features:
- **Time Series Chart**: Visualizes sales trends over time.
- **Bar Chart**: Displays total sales by ship mode.
- **Pie Chart**: Shows the distribution of sales by region.
- **Scorecards**: Highlight key metrics such as Total Sales, Total Profit, and Quantity Sold.
- **Interactive Filters**: Users can filter data by date range, product category, and customer segment.

## Purpose:
The dashboard is designed to help businesses analyze their sales performance and make data-driven decisions.

## Data Source:
The sales data is derived from Amazon sales records (sample data provided in this repository).
